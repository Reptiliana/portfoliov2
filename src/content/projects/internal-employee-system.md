## Descripción General
Este sistema fue diseñado para centralizar la gestión de permisos laborales y la generación automatizada de reportes de trabajadores, optimizando procesos internos de recursos humanos.

## Tecnologías Utilizadas
- **Frontend**: Angular
- **Backend**: .NET
- **Base de Datos**: SQL Server / MongoDB

## Puntos Clave del Desarrollo
- **Gestión de Permisos**: Desarrollo de sistemas web integrales para la administración de reportería y permisos.
- **Seguridad**: Implementación de lógica robusta de control de acceso y validación de permisos.
- **Optimización**: Mejora significativa en el flujo de permisos, reduciendo los tiempos de respuesta y aumentando la eficiencia operativa.
- **Mantenibilidad**: Refactorización de la lógica backend para eliminar procesos redundantes y mejorar la escalabilidad del sistema.
