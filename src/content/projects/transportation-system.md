## Descripción General
Plataforma web escalable para la gestión operativa de una de las principales empresas de transporte, facilitando la toma de decisiones basada en datos.

## Arquitectura de Nube
- **Frontend**: Vue.js
- **Computación**: AWS Lambda
- **Autenticación**: AWS Cognito
- **Contenedores**: Docker

## Logros Técnicos
- **Reportería en Tiempo Real**: Procesamiento eficiente de grandes volúmenes de datos para la generación de informes operativos.
- **Experiencia de Usuario**: Construcción de componentes reutilizables enfocados en la fluidez de la interfaz.
- **Optimización de Consultas**: Reducción de latencia en la consulta de información crítica del sector transporte.
- **Despliegue Moderno**: Gestión completa del ciclo de vida de la aplicación mediante contenedores.
